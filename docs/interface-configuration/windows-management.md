# Gestion automatisée des fenêtres

Gérer toutes les fenêtre, comme si elles n'étaient qu'une seule fenêtre, en {{ "un seul clic" | keyword }}.

Pour réduire toutes les fenêtres d'un seul clic, vous avez le bouton {{ "Réduire" | keywordi }} :

<figure style="text-align: center;">
    <a href="/images/interface-configuration/windoww-minimize.png" class="glightbox" data-gallery="galerie" title="Réduire toutes les fenêtres  en un seul clic">
        <img src="/images/interface-configuration/windoww-minimize.png" title="Réduire toutes les fenêtres"/>
    </a>
    <figcaption><em>Réduire toutes les fenêtres en un seul clic</em></figcaption>
</figure>

Pour agrandir toutes les fenêtres d'un seul clic :

<figure style="text-align: center;" title="Ici le tooltip">
    <a href="/images/interface-configuration/windows-maximize.png" class="glightbox" data-gallery="galerie" title="Agradir toutes les fenêtres en un seul clic">
        <img src="/images/interface-configuration/windows-maximize.png" title="gradir toutes les fenêtres"/>
    </a>
    <figcaption><em>Agradir toutes les fenêtres en un seul clic</em></figcaption>
</figure>

En réactivant la fenêtre principale {{ "Strategy Automation" | keyword }} toutes les autres fenêtres reprennent leur place sur votre écran.
