# Gestion automatisée des fenêtres

Gérer toutes les fenêtre, comme si elles n'étaient qu'une seule fenêtre, en {{ "un seul clique" | keyword }}.

Pour réduire toutes les fenêtres d'un seul clique, vous avez le bouton {{ "Réduire" | keywordi }} :

<figure style="text-align: center;" title="Ici le tooltip">
  <img src="/images/interface-configuration/windoww-minimize.png" class="noborder"/>
  <figcaption><em>Figure 1 – Le style le plus employé</em></figcaption>
</figure>

Pour agrandir toutes les fenêtres d'un seul clique :

<figure style="text-align: center;" title="Ici le tooltip">
  <img src="/images/interface-configuration/windows-maximize.png" class="noborder"/>
  <figcaption><em>Figure 1 – Le style le plus employé</em></figcaption>
</figure>

En réactivant la fenêtre principale {{ "Strategy Automation" | keyword }} toutes les autres fenêtres reprennent leur place sur votre écran.
