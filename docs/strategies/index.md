# Stratégies de trading technique

La plateforme est développées autour des {{ "stratégies automatiques de trading" | keyword }}.

Ce sont des ensembles cohérents d'indicateurs techniques intégrés dans un seul graphe pour délivrer des informations sur les cours de bourse, les tendances, des {{ "signaux d'achat et de vente" | keyword }}. Et parfrois des prédictions à l'aide des réseaux de neurones 
<a href="https://keras.io/api/" target="_blank">Keras</a>
et 
<a href="https://www.tensorflow.org/" target="_blank">Tensorflow</a>.

Certaines stratégies permettent une {{ "analyse large de la tendance" | keyword }} afin de comprendre comment évolu le cours de l'action, d'autres sont {{ "plus fines" | keyword }} pour délivrer une tendance future du cours de l'action.
